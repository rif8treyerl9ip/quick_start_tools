# CLI エイリアス ヘルプ

## 使用方法

```
COMMAND [OPTION]
```

## 説明

定型的な応答を短縮コマンドで実行できるエイリアス集。要約コマンドと回答コマンドで効率的に応答を取得できます。

## コマンド一覧

### 要約コマンド (Summary)

```
sum [N]                  指定文字数で要約
```

### 回答コマンド (Answer)

```
ans [N]                  指定文字数で回答
```

### コードコマンド (Code)

```
cod [E]                  コードで回答
```

### 理由コマンド (Reason)

```
rea [N] <content>            指定文字数 N で <content> の理由を回答

```

### 検索コマンド (Search)

```
sea [N]                  検索して指定文字数で回答
```

### 検証コマンド (Read and Verify)

```
rav <content>            <content> をスクリプト実行なしで検証
```

## 引数

```
N                        文字数指定
E                        具体例数指定
```
